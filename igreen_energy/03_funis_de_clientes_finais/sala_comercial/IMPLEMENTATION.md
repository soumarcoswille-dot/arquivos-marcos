# Guia de Implementação Técnica: Projeto Sala Comercial

Este guia foca na integração do mundo digital com a operação física da sala comercial.

## 1. Landing Page para Captura Local
A LP deve ser extremamente leve e focada em dispositivos móveis (quem está na rua vendo o carro de som ou rádio).
- **CTA Principal:** "Agendar visita na Sala" ou "Abrir no Google Maps".
- **Social Proof:** Fotos reais da fachada da sala e equipe.

## 2. Rastreamento de Conversão (Offline Tracking)
Para medir o sucesso dos R$ 5.000,00 de marketing:
- **QR Codes Únicos:** Usar um QR Code diferente para Panfletos, Carro de Som e Ads.
- **Formulário de Entrada:** Perguntar na sala "Como você nos conheceu?".

## 3. Banco de Dados de Leads (OPERACIONAL)
Estrutura simples para o consultor preencher na sala:
- Nome, Telefone, Conta de Luz (Foto), Status (Pendente/Registrado).

## 4. Próximos Passos Técnicos
- [ ] Desenvolver Landing Page responsiva.
- [ ] Configurar link de agendamento automático (Calendly ou WhatsApp Business).
- [ ] Criar arte dos Panfletos (Mockups).
- [ ] Escrever scripts para o Carro de Som e Spot de Rádio.
