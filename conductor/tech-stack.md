# Tech Stack

## Automation & Backend
- **n8n**: Workflow orchestration and routing logic.
- **OpenAI / Gemini**: AI models for message generation and lead qualification.
- **WhatsApp API**: Messaging interface (Evolution API or similar).

## Frontend & Funnels
- **HTML/CSS/JS**: Vanilla development for premium interfaces.
- **Vercel**: Deployment platform.
- **GitHub**: Version control.

## Integration
- **CRM**: Custom CRM (Rodrigo's) for lead management.
- **Webhooks**: For data transfer between funnel, n8n, and CRM.
