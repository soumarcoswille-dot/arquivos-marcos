# 🛠️ Como Utilizar o Agente Antigravity com Skills

Para aproveitar ao máximo minhas capacidades, você pode me guiar usando os nomes das habilidades listadas no catálogo. Aqui estão exemplos práticos de como "comandar" o uso dessas ferramentas.

## 1. Planejamento de Novas Funcionalidades
Antes de codificar, peça um plano:
> *"Antigravity, use a skill **writing-plans** para criar um roteiro de implementação para um sistema de login na pasta Holding."*

## 2. Garantia de Qualidade e Código Limpo
Se você quer que eu resolva um bug ou implemente algo com testes:
> *"Use o workflow de **tdd-workflow** e a skill **debugger** para encontrar e corrigir o erro no módulo de cálculos."*

## 3. SEO e Marketing Digital
Se você está trabalhando em conteúdo para a Holding ou iGreen:
> *"Faça uma auditoria de SEO na landing page atual usando a skill **seo-audit** e sugira melhorias com a skill **content-creator**."*

## 4. Design Premium (Wow Factor)
Para criar interfaces que impressionam:
> *"Crie um novo componente de card para o dashboard usando a skill **frontend-design** e adicione animações com **animejs-animation**."*

## 5. Pesquisa Profunda
Se precisar de informações sobre o mercado ou tecnologia:
> *"Use a skill **deep-research** para analisar os principais concorrentes da iGreen Energy no Sul do Brasil e crie um relatório na pasta Docs."*

---

## Dicas de Ouro 💡

- **Seja Específico:** Quanto mais contexto você der sobre *onde* (pasta) e *o quê* (funcionalidade), melhor a skill funcionará.
- **Peça Combinações:** Eu posso usar várias habilidades ao mesmo tempo. Ex: *"Use **database-design** para modelar o banco e **pydantic-models-py** para criar as classes em Python."*
- **Pergunte sobre Novas Skills:** Se não souber se tenho uma capacidade, pergunte: *"Existe alguma skill para [problema]?"*
